# Filmi2K Stremio Addon

Stremio addon за [Filmi2K.com](https://www.filmi2k.com/) - Български филми онлайн.

## Функции

- **15 каталога** с филми по категории (Екшън, Комедия, Драма, Трилър и др.)
- **Търсене** на филми
- **Метаданни** - описание, актьори, режисьор, година, жанр
- **Стрийм линкове** - автоматично извличане на видео URLs от вградените плейъри
- **Кеширане** - 1 час кеш за по-бързо зареждане

## Инсталация

### Изисквания
- [Node.js](https://nodejs.org/) v14+

### Стъпки

```bash
# 1. Инсталирай зависимостите
npm install

# 2. Стартирай addon-а
npm start
```

### Добавяне в Stremio

1. Стартирай addon-а с `npm start`
2. Отвори **Stremio**
3. Отиди в **Settings** → **Addons**
4. В полето за URL въведи: `http://localhost:7000/manifest.json`
5. Натисни **Install**

## Конфигурация

- **PORT** - Порт на сървъра (по подразбиране: 7000)
  ```bash
  PORT=8080 npm start
  ```

## Бележки

- Филмите се зареждат от filmi2k.com в реално време
- Някои стрийм източници може да не работят директно в Stremio поради DRM или anti-bot защити
- При проблеми с възпроизвеждането, филмът може да се отвори директно в браузъра
- Addon-ът кешира резултатите за 1 час за по-добра производителност
